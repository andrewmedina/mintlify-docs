<Info>
  **Voraussetzung**: Bitte installieren Sie Node.js (Version 19 oder höher) bevor Sie fortfahren.
</Info>

Befolgen Sie diese Schritte, um Mintlify auf Ihrem Betriebssystem zu installieren und auszuführen:

**Schritt 1**: Mintlify installieren:

<CodeGroup>

  ```bash npm
  npm i -g mintlify
  ```

```bash yarn
yarn global add mintlify
```

</CodeGroup>

**Schritt 2**: Navigieren Sie zum Dokumentationsverzeichnis (wo sich die `mint.json`-Datei befindet) und führen Sie den folgenden Befehl aus:

```bash
mintlify dev
```

Eine lokale Vorschau Ihrer Dokumentation wird unter `http://localhost:3000` verfügbar sein.

### Benutzerdefinierte Ports

Standardmäßig verwendet Mintlify Port 3000. Sie können den Port, auf dem Mintlify läuft, mit dem Flag `--port` anpassen. Um Mintlify beispielsweise auf Port 3333 auszuführen, verwenden Sie diesen Befehl:

```bash
mintlify dev --port 3333
```

Wenn Sie versuchen, Mintlify auf einem Port auszuführen, der bereits belegt ist, wird der nächste verfügbare Port verwendet:

```md
Port 3000 is already in use. Trying 3001 instead.
```

## Mintlify Versionen

Bitte beachten Sie, dass jede CLI-Version mit einer bestimmten Version von Mintlify verbunden ist. Wenn Ihre lokale Website nicht mit der Produktionsversion übereinstimmt, aktualisieren Sie bitte die CLI:

<CodeGroup>

```bash npm
npm i -g mintlify@latest
```

```bash yarn
yarn global upgrade mintlify
```

</CodeGroup>

## Links überprüfen

Die CLI kann bei der Validierung von Referenzlinks in Ihrer Dokumentation helfen. Um fehlerhafte Links zu identifizieren, verwenden Sie den folgenden Befehl:

```bash
mintlify broken-links
```

## Fehlerbehebung

<AccordionGroup>
  <Accordion title='Fehler: Konnte das "sharp"-Modul mit der darwin-arm64 Laufzeitumgebung nicht laden'>

    Dies kann an einer veralteten Version von Node liegen. Versuchen Sie Folgendes:
    1. Entfernen Sie die aktuell installierte Version von Mintlify: `npm remove -g mintlify`
    2. Aktualisieren Sie auf Node v19 oder höher.
    3. Installieren Sie Mintlify neu: `npm install -g mintlify`
  </Accordion>

  <Accordion title="Problem: Unbekannter Fehler tritt auf">
  
    Lösung: Gehen Sie zum Hauptverzeichnis Ihres Geräts und löschen Sie den Ordner \~/.mintlify. Führen Sie danach `mintlify dev` erneut aus.
  </Accordion>
</AccordionGroup>
