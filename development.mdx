Volg deze stappen om Mintlify op je besturingssysteem te installeren en uit te voeren:

**Stap 1**: Installeer Mintlify:

<CodeGroup>

  ```bash npm
  npm i -g mintlify
  ```

```bash yarn
yarn global add mintlify
```

</CodeGroup>

**Stap 2**: Navigeer naar de documentatie directory (waar het `mint.json` bestand zich bevindt) en voer het volgende commando uit:

```bash
mintlify dev
```

Een lokale preview van je documentatie zal beschikbaar zijn op `http://localhost:3000`.

### Aangepaste Poorten

Standaard gebruikt Mintlify poort 3000. Je kunt de poort waarop Mintlify draait aanpassen met de `--port` flag. Om Mintlify bijvoorbeeld op poort 3333 te draaien, gebruik je dit commando:

```bash
mintlify dev --port 3333
```

Als je probeert Mintlify te draaien op een poort die al in gebruik is, zal het de eerstvolgende beschikbare poort gebruiken:

```md
Poort 3000 is al in gebruik. Probeer 3001 in plaats daarvan.
```

## Mintlify Versies

Let op dat elke CLI-release is gekoppeld aan een specifieke versie van Mintlify. Als je lokale website niet overeenkomt met de productieversie, update dan de CLI:

<CodeGroup>

```bash npm
npm i -g mintlify@latest
```

```bash yarn
yarn global upgrade mintlify
```

</CodeGroup>

## Links Valideren

De CLI kan helpen bij het valideren van referentielinks in je documentatie. Om eventuele verbroken links te identificeren, gebruik je het volgende commando:

```bash
mintlify broken-links
```
