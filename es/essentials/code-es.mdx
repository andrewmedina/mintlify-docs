## Básico

### Código en Línea

Para denotar una `palabra` o `frase` como código, enciérrala entre acentos graves (`).

```
Para denotar una `palabra` o `frase` como código, enciérrala entre acentos graves (`).
```

### Bloque de Código

Utiliza [bloques de código delimitados](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) encerrando el código entre tres acentos graves y coloca el lenguaje de programación de tu fragmento después de los acentos iniciales para obtener resaltado de sintaxis. Opcionalmente, también puedes escribir el nombre de tu código después del lenguaje de programación.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
