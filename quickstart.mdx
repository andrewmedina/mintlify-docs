## Configure seu ambiente de desenvolvimento

Aprenda como atualizar sua documentação localmente e implantá-la publicamente.

### Edite e visualize

<AccordionGroup>
  <Accordion icon="github" title="Clone sua documentação localmente">
    Durante o processo de integração, criamos um repositório no seu Github com
    o conteúdo da sua documentação. Você pode encontrar este repositório em nosso
    [painel](https://dashboard.mintlify.com). Para clonar o repositório
    localmente, siga estas
    [instruções](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository)
    no seu terminal.
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Visualize as alterações">
    A visualização ajuda você a garantir que suas alterações apareçam conforme o planejado. Desenvolvemos uma
    interface de linha de comando para renderizar essas alterações localmente. 1. Instale o
    [Mintlify CLI](https://www.npmjs.com/package/mintlify) para visualizar as
    alterações da documentação localmente com este comando: ``` npm i -g mintlify ```
    2. Execute o seguinte comando na raiz da sua documentação (onde
    `mint.json` está): ``` mintlify dev ```
  </Accordion>
</AccordionGroup>

### Implante suas alterações

<AccordionGroup>

<Accordion icon="message-bot" title="Instale nosso aplicativo do Github">
  Nosso aplicativo do Github implanta automaticamente suas alterações no site da sua documentação,
  então você não precisa gerenciar as implantações manualmente. Você pode encontrar o link para instalar em
  seu [painel](https://dashboard.mintlify.com). Depois que o bot for
  instalado com sucesso, deverá haver uma marca de verificação ao lado do hash do commit
  do repositório.
</Accordion>
<Accordion icon="rocket" title="Envie suas alterações">
  [Faça commit e envie suas alterações para o
  Git](https://docs.github.com/en/get-started/using-git/pushing-commits-to-a-remote-repository#about-git-push)
  para que suas alterações sejam atualizadas no site da sua documentação. Se você enviar e não ver que
  o aplicativo do Github implantou suas alterações com sucesso, você também pode
  atualizar sua documentação manualmente através do nosso [painel](https://dashboard.mintlify.com).
</Accordion>

</AccordionGroup>
