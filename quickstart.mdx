## Stel je ontwikkelomgeving in

Leer hoe je je documenten lokaal kunt bijwerken en ze publiek kunt maken.

### Bewerken en voorvertonen

<AccordionGroup>
  <Accordion icon="github" title="Clone je documenten lokaal">
    Tijdens het onboardingsproces hebben we een repository op je Github gemaakt met
    je documentatie-inhoud. Je kunt deze repository vinden op ons
    [dashboard](https://dashboard.mintlify.com). Om de repository
    lokaal te klonen, volg je deze
    [instructies](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository)
    in je terminal.
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Bekijk wijzigingen">
    Voorvertonen helpt je om zeker te weten dat je wijzigingen eruitzien zoals bedoeld. We hebben een
    command line interface gebouwd om deze wijzigingen lokaal weer te geven. 1. Installeer de
    [Mintlify CLI](https://www.npmjs.com/package/mintlify) om de
    documentatie-wijzigingen lokaal te bekijken met dit commando: ``` npm i -g mintlify ```
    2. Voer het volgende commando uit in de hoofdmap van je documentatie (waar
    `mint.json` staat): ``` mintlify dev ```
  </Accordion>
</AccordionGroup>

### Deploy je wijzigingen

<AccordionGroup>

<Accordion icon="message-bot" title="Installeer onze Github app">
  Onze Github app implementeert automatisch je wijzigingen in je documentatiesite, zodat je
  geen deployments zelf hoeft te beheren. Je kunt de link om te installeren vinden op
  je [dashboard](https://dashboard.mintlify.com). Zodra de bot succesvol is
  geïnstalleerd, zou er een vinkje moeten staan naast de commit hash
  van de repo.
</Accordion>
<Accordion icon="rocket" title="Push je wijzigingen">
  [Commit en push je wijzigingen naar
  Git](https://docs.github.com/en/get-started/using-git/pushing-commits-to-a-remote-repository#about-git-push)
  om je wijzigingen in je documentatiesite bij te werken. Als je pusht en niet ziet dat
  de Github app je wijzigingen succesvol heeft geïmplementeerd, kun je ook handmatig
  je documentatie bijwerken via ons [dashboard](https://dashboard.mintlify.com).
</Accordion>

</AccordionGroup>
