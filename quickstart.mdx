## Nyiapake pangembangan sampeyan

Sinau carane nganyari dokumentasi kanthi lokal lan ngatur kanggo umum.

### Nyunting lan pratinjau

<AccordionGroup>
  <Accordion icon="github" title="Nglone dokumentasi sampeyan sacara lokal">
    Nalika proses onboarding, kita nggawe repository ing Github sampeyan karo
    isi dokumentasi. Sampeyan bisa nemokake repository iki ing
    [dashboard](https://dashboard.mintlify.com). Kanggo nglone repository
    sacara lokal, tutna [pandhuan](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository)
    iki ing terminal sampeyan.
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Ndeleng owahan">
    Ndeleng owahan mbantu sampeyan mastekake owahan sing ditindakake katon kaya sing dikarepake. Kita mbangun
    command line interface kanggo nampilake owahan iki sacara lokal. 1. Install
    [Mintlify CLI](https://www.npmjs.com/package/mintlify) kanggo ndeleng
    owahan dokumentasi sacara lokal nganggo printah iki: ``` npm i -g mintlify ```
    2. Jalanake printah ing ngisor iki ing root dokumentasi sampeyan (ing ngendi
    `mint.json` ana): ``` mintlify dev ```
  </Accordion>
</AccordionGroup>

### Ngetrapake owahan sampeyan

<AccordionGroup>

<Accordion icon="message-bot" title="Install aplikasi Github kita">
  Aplikasi Github kita otomatis ngetrapake owahan sampeyan menyang situs dokumentasi,
  dadi sampeyan ora perlu ngatur penerapan dhewe. Sampeyan bisa nemokake link kanggo
  install ing [dashboard](https://dashboard.mintlify.com). Sawise bot kasil
  diinstal, kudu ana tandha centang ing cedhak commit hash
  repository.
</Accordion>
<Accordion icon="rocket" title="Push owahan sampeyan">
  [Commit lan push owahan sampeyan menyang
  Git](https://docs.github.com/en/get-started/using-git/pushing-commits-to-a-remote-repository#about-git-push)
  supaya owahan sampeyan dianyari ing situs dokumentasi. Yen sampeyan push lan ora weruh
  yen aplikasi Github kasil ngetrapake owahan sampeyan, sampeyan uga bisa manual
  nganyari dokumentasi liwat [dashboard](https://dashboard.mintlify.com).
</Accordion>

</AccordionGroup>
