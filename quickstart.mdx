## Entwicklungsumgebung einrichten

Lernen Sie, wie Sie Ihre Dokumentation lokal aktualisieren und öffentlich bereitstellen können.

### Bearbeiten und Vorschau

<AccordionGroup>
  <Accordion icon="github" title="Klonen Sie Ihre Dokumentation lokal">
    Während des Onboarding-Prozesses haben wir ein Repository auf Ihrem Github mit
    Ihren Dokumentationsinhalten erstellt. Sie finden dieses Repository in unserem
    [Dashboard](https://dashboard.mintlify.com). Um das Repository
    lokal zu klonen, folgen Sie diesen
    [Anweisungen](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository)
    in Ihrem Terminal.
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Vorschau der Änderungen">
    Die Vorschau hilft Ihnen sicherzustellen, dass Ihre Änderungen wie beabsichtigt aussehen. Wir haben eine
    Kommandozeilenschnittstelle entwickelt, um diese Änderungen lokal anzuzeigen. 1. Installieren Sie die
    [Mintlify CLI](https://www.npmjs.com/package/mintlify), um die
    Dokumentationsänderungen lokal mit diesem Befehl anzuzeigen: ``` npm i -g mintlify ```
    2. Führen Sie den folgenden Befehl im Hauptverzeichnis Ihrer Dokumentation aus (wo sich
    `mint.json` befindet): ``` mintlify dev ```
  </Accordion>
</AccordionGroup>

### Änderungen bereitstellen

<AccordionGroup>

<Accordion icon="message-bot" title="Installieren Sie unsere Github-App">
  Unsere Github-App stellt Ihre Änderungen automatisch auf Ihrer Dokumentationsseite bereit, sodass
  Sie die Bereitstellungen nicht selbst verwalten müssen. Den Installationslink finden Sie in
  Ihrem [Dashboard](https://dashboard.mintlify.com). Sobald der Bot erfolgreich
  installiert wurde, sollte neben dem Commit-Hash des Repos ein Häkchen
  erscheinen.
</Accordion>
<Accordion icon="rocket" title="Pushen Sie Ihre Änderungen">
  [Committen und pushen Sie Ihre Änderungen zu
  Git](https://docs.github.com/en/get-started/using-git/pushing-commits-to-a-remote-repository#about-git-push),
  damit Ihre Änderungen auf Ihrer Dokumentationsseite aktualisiert werden. Wenn Sie pushen und nicht sehen,
  dass die Github-App Ihre Änderungen erfolgreich bereitgestellt hat, können Sie Ihre
  Dokumentation auch manuell über unser [Dashboard](https://dashboard.mintlify.com) aktualisieren.
</Accordion>

</AccordionGroup>
