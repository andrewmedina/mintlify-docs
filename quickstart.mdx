## Configura tu desarrollo

Aprende cómo actualizar tu documentación localmente y desplegarla al público.

### Editar y previsualizar

<AccordionGroup>
  <Accordion icon="github" title="Clona tu documentación localmente">
    Durante el proceso de incorporación, creamos un repositorio en tu Github con
    el contenido de tu documentación. Puedes encontrar este repositorio en nuestro
    [panel de control](https://dashboard.mintlify.com). Para clonar el repositorio
    localmente, sigue estas
    [instrucciones](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository)
    en tu terminal.
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Previsualiza los cambios">
    La previsualización te ayuda a asegurarte de que tus cambios se vean como lo deseas. Hemos construido una
    interfaz de línea de comandos para renderizar estos cambios localmente. 1. Instala el
    [CLI de Mintlify](https://www.npmjs.com/package/mintlify) para previsualizar los
    cambios en la documentación localmente con este comando: ``` npm i -g mintlify ```
    2. Ejecuta el siguiente comando en la raíz de tu documentación (donde
    está `mint.json`): ``` mintlify dev ```
  </Accordion>
</AccordionGroup>

### Despliega tus cambios

<AccordionGroup>

<Accordion icon="message-bot" title="Instala nuestra aplicación de Github">
  Nuestra aplicación de Github despliega automáticamente tus cambios en el sitio de tu documentación,
  por lo que no necesitas gestionar los despliegues por tu cuenta. Puedes encontrar el enlace para instalar en
  tu [panel de control](https://dashboard.mintlify.com). Una vez que el bot haya sido
  instalado exitosamente, debería aparecer una marca de verificación junto al hash del commit
  del repositorio.
</Accordion>
<Accordion icon="rocket" title="Sube tus cambios">
  [Confirma y sube tus cambios a
  Git](https://docs.github.com/en/get-started/using-git/pushing-commits-to-a-remote-repository#about-git-push)
  para que tus cambios se actualicen en el sitio de tu documentación. Si subes los cambios y no ves que
  la aplicación de Github haya desplegado exitosamente tus cambios, también puedes actualizar
  manualmente tu documentación a través de nuestro [panel de control](https://dashboard.mintlify.com).
</Accordion>

</AccordionGroup>
