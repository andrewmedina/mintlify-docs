## Afbeeldingen

### Markdown Gebruiken

De [markdown syntax](https://www.markdownguide.org/basic-syntax/#images) laat je afbeeldingen toevoegen met de volgende code

```md
![titel](/path/image.jpg)
```

Let op dat de bestandsgrootte van de afbeelding kleiner moet zijn dan 5MB. Anders raden we aan om de afbeelding te hosten op een dienst zoals [Cloudinary](https://cloudinary.com/) of [S3](https://aws.amazon.com/s3/). Je kunt dan die URL gebruiken en insluiten.

### Insluitingen Gebruiken

Voor meer aanpassingsmogelijkheden met afbeeldingen kun je ook [insluitingen](/writing-content/embed) gebruiken om afbeeldingen toe te voegen

```html
<img height="200" src="/path/image.jpg" />
```

<Tip>

Mintlify ondersteunt [HTML-tags in Markdown](https://www.markdownguide.org/basic-syntax/#html). Dit is handig als je de voorkeur geeft aan HTML-tags boven Markdown-syntax, en het stelt je in staat om documentatie te maken met oneindige flexibiliteit.

</Tip>

### iFrames

Laadt een andere HTML-pagina binnen het document. Wordt meestal gebruikt voor het insluiten van video's.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
