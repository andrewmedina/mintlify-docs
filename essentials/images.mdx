## Imagem

### Usando Markdown

A [sintaxe markdown](https://www.markdownguide.org/basic-syntax/#images) permite adicionar imagens usando o seguinte código

```md
![título](/path/image.jpg)
```

Observe que o tamanho do arquivo de imagem deve ser menor que 5MB. Caso contrário, recomendamos hospedar em um serviço como [Cloudinary](https://cloudinary.com/) ou [S3](https://aws.amazon.com/s3/). Você pode então usar essa URL e incorporar.

### Usando Incorporações

Para obter mais personalização com imagens, você também pode usar [incorporações](/writing-content/embed) para adicionar imagens

```html
<img height="200" src="/path/image.jpg" />
```

### iFrames

Carrega outra página HTML dentro do documento. Mais comumente usado para incorporar vídeos.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
