Het navigatiemenu is de lijst met links op elke website.

Je zult waarschijnlijk `mint.json` bijwerken elke keer dat je een nieuwe pagina toevoegt. Pagina's verschijnen niet automatisch.

## Navigatie syntax

Onze navigatie syntax is recursief, wat betekent dat je geneste navigatiegroepen kunt maken. Je hoeft `.mdx` niet op te nemen in paginanamen.

<CodeGroup>

```json Normale Navigatie
"navigation": [
    {
        "group": "Getting Started",
        "pages": ["quickstart"]
    }
]
```

```json Geneste Navigatie
"navigation": [
    {
        "group": "Getting Started",
        "pages": [
            "quickstart",
            {
                "group": "Nested Reference Pages",
                "pages": ["nested-reference-page"]
            }
        ]
    }
]
```

</CodeGroup>

## Mappen

Plaats je MDX-bestanden eenvoudig in mappen en werk de paden bij in `mint.json`.

Bijvoorbeeld, om een pagina te hebben op `https://yoursite.com/your-folder/your-page` zou je een map maken met de naam `your-folder` met daarin een MDX-bestand genaamd `your-page.mdx`.

<Warning>

Je kunt `api` niet gebruiken als mapnaam tenzij je deze in een andere map nest. Mintlify gebruikt Next.js, dat de `api`-map op het hoogste niveau reserveert voor interne serveraanroepen. Een mapnaam zoals `api-reference` zou wel worden geaccepteerd.

</Warning>

```json Navigatie Met Map
"navigation": [
    {
        "group": "Groepsnaam",
        "pages": ["your-folder/your-page"]
    }
]
```

## Verborgen Pagina's

MDX-bestanden die niet zijn opgenomen in `mint.json` verschijnen niet in de zijbalk maar zijn toegankelijk via de zoekbalk en door er direct naartoe te linken.
