ナビゲーションメニューは、すべてのウェブサイトにあるリンクのリストです。

新しいページを追加するたびに、`mint.json`を更新する必要があります。ページは自動的には表示されません。

## ナビゲーション構文

ナビゲーション構文は再帰的であり、ネストされたナビゲーショングループを作成できます。ページ名に`.mdx`を含める必要はありません。

<CodeGroup>

```json Regular Navigation
"navigation": [
    {
        "group": "Getting Started",
        "pages": ["quickstart"]
    }
]
```

```json Nested Navigation
"navigation": [
    {
        "group": "Getting Started",
        "pages": [
            "quickstart",
            {
                "group": "Nested Reference Pages",
                "pages": ["nested-reference-page"]
            }
        ]
    }
]
```

</CodeGroup>

## フォルダ

MDXファイルをフォルダに配置し、`mint.json`内のパスを更新するだけです。

例えば、`https://yoursite.com/your-folder/your-page`にページを作成するには、`your-folder`というフォルダを作成し、その中に`your-page.mdx`というMDXファイルを配置します。

<Warning>

別のフォルダ内にネストしない限り、フォルダ名として`api`を使用することはできません。MintlifyはNext.jsを使用しており、トップレベルの`api`フォルダは内部サーバー呼び出し用に予約されています。`api-reference`のようなフォルダ名は使用可能です。

</Warning>

```json Navigation With Folder
"navigation": [
    {
        "group": "Group Name",
        "pages": ["your-folder/your-page"]
    }
]
```

## 非表示ページ

`mint.json`に含まれていないMDXファイルはサイドバーには表示されませんが、検索バーから、また直接リンクすることでアクセス可能です。
