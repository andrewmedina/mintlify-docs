Menu navigasi yaiku daftar pranala ing saben situs web.

Sampeyan mesthine bakal nganyari `mint.json` saben nambah kaca anyar. Kaca ora otomatis ditampilake.

## Sintaks Navigasi

Sintaks navigasi kita rekursif sing tegese sampeyan bisa nggawe kelompok navigasi sing kasusun. Sampeyan ora perlu nyertakake `.mdx` ing jeneng kaca.

<CodeGroup>

```json Regular Navigation
"navigation": [
    {
        "group": "Getting Started",
        "pages": ["quickstart"]
    }
]
```

```json Nested Navigation
"navigation": [
    {
        "group": "Getting Started",
        "pages": [
            "quickstart",
            {
                "group": "Nested Reference Pages",
                "pages": ["nested-reference-page"]
            }
        ]
    }
]
```

</CodeGroup>

## Folder

Cukup lebokake berkas MDX ing folder lan nganyari jalur ing `mint.json`.

Contone, kanggo duwe kaca ing `https://yoursite.com/your-folder/your-page` sampeyan kudu nggawe folder sing jenenge `your-folder` sing ngemot berkas MDX sing jenenge `your-page.mdx`.

<Warning>

Sampeyan ora bisa nggunakake `api` kanggo jeneng folder kajaba sampeyan ndlesepake ing folder liya. Mintlify nggunakake Next.js sing nyimpen folder `api` tingkat puncak kanggo panggilan server internal. Jeneng folder kayata `api-reference` bakal ditampa.

</Warning>

```json Navigation With Folder
"navigation": [
    {
        "group": "Group Name",
        "pages": ["your-folder/your-page"]
    }
]
```

## Kaca Kadhelikake

Berkas MDX sing ora kalebu ing `mint.json` ora bakal ditampilake ing sidebar nanging bisa diakses liwat bilah telusuran lan kanthi langsung menyang pranala kasebut.
