Das Navigationsmenü ist die Liste von Links auf jeder Website.

Sie werden `mint.json` wahrscheinlich jedes Mal aktualisieren müssen, wenn Sie eine neue Seite hinzufügen. Seiten erscheinen nicht automatisch.

## Navigationssyntax

Unsere Navigationssyntax ist rekursiv, was bedeutet, dass Sie verschachtelte Navigationsgruppen erstellen können. Sie müssen `.mdx` nicht in Seitennamen einschließen.

<CodeGroup>

```json Regular Navigation
"navigation": [
    {
        "group": "Getting Started",
        "pages": ["quickstart"]
    }
]
```

```json Nested Navigation
"navigation": [
    {
        "group": "Getting Started",
        "pages": [
            "quickstart",
            {
                "group": "Nested Reference Pages",
                "pages": ["nested-reference-page"]
            }
        ]
    }
]
```

</CodeGroup>

## Ordner

Legen Sie einfach Ihre MDX-Dateien in Ordner und aktualisieren Sie die Pfade in `mint.json`.

Um zum Beispiel eine Seite unter `https://yoursite.com/your-folder/your-page` zu haben, würden Sie einen Ordner namens `your-folder` erstellen, der eine MDX-Datei namens `your-page.mdx` enthält.

<Warning>

Sie können `api` nicht als Ordnernamen verwenden, es sei denn, Sie verschachteln ihn in einem anderen Ordner. Mintlify verwendet Next.js, das den `api`-Ordner auf oberster Ebene für interne Serveraufrufe reserviert. Ein Ordnername wie `api-reference` wäre zulässig.

</Warning>

```json Navigation With Folder
"navigation": [
    {
        "group": "Group Name",
        "pages": ["your-folder/your-page"]
    }
]
```

## Versteckte Seiten

MDX-Dateien, die nicht in `mint.json` enthalten sind, werden in der Seitenleiste nicht angezeigt, sind aber über die Suchleiste und durch direktes Verlinken zugänglich.
