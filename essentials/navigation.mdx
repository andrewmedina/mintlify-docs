O menu de navegação é a lista de links em todo o site.

Você provavelmente atualizará o `mint.json` sempre que adicionar uma nova página. As páginas não aparecem automaticamente.

## Sintaxe de navegação

Nossa sintaxe de navegação é recursiva, o que significa que você pode criar grupos de navegação aninhados. Você não precisa incluir `.mdx` nos nomes das páginas.

<CodeGroup>

```json Navegação Regular
"navigation": [
    {
        "group": "Getting Started",
        "pages": ["quickstart"]
    }
]
```

```json Navegação Aninhada
"navigation": [
    {
        "group": "Getting Started",
        "pages": [
            "quickstart",
            {
                "group": "Nested Reference Pages",
                "pages": ["nested-reference-page"]
            }
        ]
    }
]
```

</CodeGroup>

## Pastas

Simplesmente coloque seus arquivos MDX em pastas e atualize os caminhos no `mint.json`.

Por exemplo, para ter uma página em `https://yoursite.com/your-folder/your-page`, você criaria uma pasta chamada `your-folder` contendo um arquivo MDX chamado `your-page.mdx`.

<Warning>

Você não pode usar `api` como nome de uma pasta, a menos que a coloque dentro de outra pasta. O Mintlify usa Next.js, que reserva a pasta `api` de nível superior para chamadas internas do servidor. Um nome de pasta como `api-reference` seria aceito.

</Warning>

```json Navegação Com Pasta
"navigation": [
    {
        "group": "Group Name",
        "pages": ["your-folder/your-page"]
    }
]
```

## Páginas Ocultas

Arquivos MDX não incluídos no `mint.json` não aparecerão na barra lateral, mas são acessíveis através da barra de pesquisa e linking direto para eles.
