## Títulos

Melhor utilizado para cabeçalhos de seção.

```md
## Títulos
```

### Subtítulos

Melhor utilizado para cabeçalhos de subseção.

```md
### Subtítulos
```

<Tip>

Cada **título** e **subtítulo** cria uma âncora e também aparece na tabela de conteúdo à direita.

</Tip>

## Formatação de Texto

Suportamos a maioria das formatações markdown. Simplesmente adicione `**`, `_`, ou `~` em volta do texto para formatá-lo.

| Estilo        | Como escrever     | Resultado       |
| ------------- | ----------------- | --------------- |
| Negrito       | `**negrito**`     | **negrito**     |
| Itálico       | `_itálico_`       | _itálico_       |
| Tachado       | `~tachado~`       | ~tachado~       |

Você pode combinar estes estilos. Por exemplo, escreva `**_negrito e itálico_**` para obter texto em **_negrito e itálico_**.

Você precisa usar HTML para escrever texto sobrescrito e subscrito. Isto é, adicione `<sup>` ou `<sub>` em volta do seu texto.

| Tamanho do Texto | Como escrever            | Resultado             |
| --------------- | ------------------------ | --------------------- |
| Sobrescrito     | `<sup>sobrescrito</sup>` | <sup>sobrescrito</sup> |
| Subscrito       | `<sub>subscrito</sub>`   | <sub>subscrito</sub>   |

## Vinculando Páginas

Você pode adicionar um link envolvendo texto em `[]()`. Você escreveria `[link para o google](https://google.com)` para criar um [link para o google](https://google.com).

Links para páginas em sua documentação precisam ser relativos à raiz. Basicamente, você deve incluir todo o caminho da pasta. Por exemplo, `[link para texto](/writing-content/text)` vincula à página "Texto" em nossa seção de componentes.

Links relativos como `[link para texto](../text)` abrirão mais lentamente porque não podemos otimizá-los tão facilmente.

## Citações em Bloco

### Linha Única

Para criar uma citação em bloco, adicione um `>` na frente de um parágrafo.

> Dorothy a seguiu por muitas das belas salas em seu castelo.

```md
> Dorothy a seguiu por muitas das belas salas em seu castelo.
```

### Múltiplas Linhas

> Dorothy a seguiu por muitas das belas salas em seu castelo.
>
> A Bruxa ordenou que ela limpasse as panelas e chaleiras, varresse o chão e mantivesse o fogo alimentado com lenha.

```md
> Dorothy a seguiu por muitas das belas salas em seu castelo.
>
> A Bruxa ordenou que ela limpasse as panelas e chaleiras, varresse o chão e mantivesse o fogo alimentado com lenha.
```

### LaTeX

<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>

```md
<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>
```
