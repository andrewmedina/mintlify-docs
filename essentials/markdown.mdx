## タイトル

セクションヘッダーに最適です。

```md
## タイトル
```

### サブタイトル

サブセクションヘッダーに最適です。

```md
### サブタイトル
```

<Tip>

各**タイトル**と**サブタイトル**はアンカーを作成し、右側の目次にも表示されます。

</Tip>

## テキストフォーマット

ほとんどのマークダウンフォーマットをサポートしています。テキストの周りに`**`、`_`、`~`を追加するだけでフォーマットできます。

| スタイル | 記述方法 | 結果 |
| ------------- | ----------------- | --------------- |
| 太字 | `**太字**` | **太字** |
| イタリック | `_イタリック_` | _イタリック_ |
| 取り消し線 | `~取り消し線~` | ~取り消し線~ |

これらを組み合わせることができます。例えば、`**_太字とイタリック_**`と書くと、**_太字とイタリック_**のテキストになります。

上付き文字と下付き文字を書くにはHTMLを使用する必要があります。テキストの周りに`<sup>`または`<sub>`を追加します。

| テキストサイズ | 記述方法 | 結果 |
| ----------- | ------------------------ | ---------------------- |
| 上付き文字 | `<sup>上付き文字</sup>` | <sup>上付き文字</sup> |
| 下付き文字 | `<sub>下付き文字</sub>` | <sub>下付き文字</sub> |

## ページへのリンク

`[]()`でテキストを囲むことでリンクを追加できます。`[Googleへのリンク](https://google.com)`と書くと、[Googleへのリンク](https://google.com)になります。

ドキュメント内のページへのリンクはルート相対パスである必要があります。基本的に、フォルダパス全体を含める必要があります。例えば、`[テキストへのリンク](/writing-content/text)`はコンポーネントセクションの「テキスト」ページにリンクします。

`[テキストへのリンク](../text)`のような相対リンクは、最適化が容易ではないため、開くのに時間がかかります。

## ブロッククォート

### 1行

ブロッククォートを作成するには、段落の前に`>`を追加します。

> ドロシーは彼女の城の多くの美しい部屋を通って彼女に従いました。

```md
> ドロシーは彼女の城の多くの美しい部屋を通って彼女に従いました。
```

### 複数行

> ドロシーは彼女の城の多くの美しい部屋を通って彼女に従いました。
>
> 魔女は彼女に鍋や釜を洗い、床を掃き、薪で火を絶やさないように命じました。

```md
> ドロシーは彼女の城の多くの美しい部屋を通って彼女に従いました。
>
> 魔女は彼女に鍋や釜を洗い、床を掃き、薪で火を絶やさないように命じました。
```

### LaTeX

MintlifyはLatexコンポーネントを通じて[LaTeX](https://www.latex-project.org)をサポートしています。

<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>

```md
<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>
```
