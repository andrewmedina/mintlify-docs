## 标题

最适合用作章节标题。

```md
## 标题
```

### 副标题

最适合用作小节标题。

```md
### 副标题
```

<Tip>

每个**标题**和**副标题**都会创建一个锚点，并且会显示在右侧的目录中。

</Tip>

## 文本格式

我们支持大多数 markdown 格式。只需在文本周围添加 `**`、`_` 或 `~` 即可设置格式。

| 样式     | 如何编写        | 效果           |
| -------- | -------------- | -------------- |
| 粗体     | `**粗体**`     | **粗体**       |
| 斜体     | `_斜体_`       | _斜体_         |
| 删除线   | `~删除线~`     | ~删除线~       |

你可以组合使用这些格式。例如，写 `**_粗体和斜体_**` 可以得到 **_粗体和斜体_** 文本。

要写上标和下标文本，你需要使用 HTML。即在文本周围添加 `<sup>` 或 `<sub>`。

| 文本大小 | 如何编写              | 效果                   |
| -------- | -------------------- | --------------------- |
| 上标     | `<sup>上标</sup>`    | <sup>上标</sup>       |
| 下标     | `<sub>下标</sub>`    | <sub>下标</sub>       |

## 链接到页面

你可以通过将文本包装在 `[]()` 中来添加链接。例如，写 `[链接到谷歌](https://google.com)` 来创建 [链接到谷歌](https://google.com)。

链接到文档中的页面需要使用根相对路径。基本上，你应该包含完整的文件夹路径。例如，`[链接到文本](/writing-content/text)` 链接到我们组件部分中的"文本"页面。

相对链接如 `[链接到文本](../text)` 的加载速度会更慢，因为我们无法轻易地优化它们。

## 引用块

### 单行引用

要创建引用块，在段落前面添加 `>`。

> 多萝西跟随她参观了城堡中许多美丽的房间。

```md
> 多萝西跟随她参观了城堡中许多美丽的房间。
```

### 多行引用

> 多萝西跟随她参观了城堡中许多美丽的房间。
>
> 女巫命令她清洗锅具和壶具，打扫地板，并用木柴保持火势。

```md
> 多萝西跟随她参观了城堡中许多美丽的房间。
>
> 女巫命令她清洗锅具和壶具，打扫地板，并用木柴保持火势。
```

### LaTeX

Mintlify 通过 Latex 组件支持 [LaTeX](https://www.latex-project.org)。

<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>

```md
<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>
```
