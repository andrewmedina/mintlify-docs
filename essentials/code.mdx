## Básico

### Código en Línea

Para denotar una `palabra` o `frase` como código, enciérrela entre comillas invertidas (`).

```
Para denotar una `palabra` o `frase` como código, enciérrela entre comillas invertidas (`).
```

### Bloque de Código

Utilice [bloques de código cercados](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) encerrando el código entre tres comillas invertidas y coloque el lenguaje de programación de su fragmento después de las comillas iniciales para obtener resaltado de sintaxis. Opcionalmente, también puede escribir el nombre de su código después del lenguaje de programación.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
