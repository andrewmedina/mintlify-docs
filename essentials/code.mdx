## 基本

### インラインコード

`単語`や`フレーズ`をコードとして表示するには、バッククォート(`)で囲みます。

```
`単語`や`フレーズ`をコードとして表示するには、バッククォート(`)で囲みます。
```

### コードブロック

[フェンスで囲まれたコードブロック](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks)を使用するには、3つのバッククォートでコードを囲み、先頭のバッククォートの後にプログラミング言語を記述することで、シンタックスハイライトが適用されます。オプションとして、プログラミング言語の後にコードの名前を記述することもできます。

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
