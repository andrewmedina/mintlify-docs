## Basis

### Inline Code

Om een `word` of `phrase` als code aan te duiden, plaats je het tussen backticks (`).

```
Om een `word` of `phrase` als code aan te duiden, plaats je het tussen backticks (`).
```

### Codeblok

Gebruik [omheinde codeblokken](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) door code tussen drie backticks te plaatsen en plaats de programmeertaal van je fragment na de eerste backticks om syntaxismarkering te krijgen. Optioneel kun je ook de naam van je code na de programmeertaal schrijven.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
