## Grundlagen

### Inline-Code

Um ein `Wort` oder eine `Phrase` als Code zu kennzeichnen, setzen Sie es in Backticks (`).

```
Um ein `Wort` oder eine `Phrase` als Code zu kennzeichnen, setzen Sie es in Backticks (`).
```

### Code-Block

Verwenden Sie [umschlossene Code-Blöcke](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks), indem Sie den Code in drei Backticks einschließen und den Backticks die Programmiersprache Ihres Snippets folgen lassen, um Syntaxhervorhebung zu erhalten. Optional können Sie auch den Namen Ihres Codes nach der Programmiersprache angeben.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
