## 基础

### 内联代码

要将`单词`或`短语`表示为代码，请将其括在反引号(`)中。

```
要将`单词`或`短语`表示为代码，请将其括在反引号(`)中。
```

### 代码块

使用[围栏式代码块](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks)，方法是将代码括在三个反引号中，并在开头的反引号后面注明代码段的编程语言，以获得语法高亮显示。您也可以选择在编程语言后写入代码的名称。

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
