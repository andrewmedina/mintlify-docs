## 基础

### 行内代码

要将`word`或`phrase`表示为代码，请将其用反引号(`)括起来。

```
要将`word`或`phrase`表示为代码，请将其用反引号(`)括起来。
```

### 代码块

使用[围栏式代码块](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks)，方法是将代码用三个反引号括起来，并在开头的反引号后面标注代码段的编程语言，以获得语法高亮。您还可以选择在编程语言后面写上代码的名称。

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
